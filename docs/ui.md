# 🎵 Harmonix - A unified cross-source music player for desktop — Spotify, YouTube Music, and local files in one place.

A modern, visually immersive music player web application designed to deliver an exceptional listening experience through dynamic visuals, intuitive navigation, interactive feedback, and seamless media controls.

![Harmonix Music Player](./assets/music-player-preview.png)

---

## ✨ Features

### 🎧 Immersive Listening Experience

* Dynamic album-based theming
* Animated gradient backgrounds
* Real-time audio visualizer
* Full-screen listening mode

### 🧭 Intuitive Navigation

* Clean sidebar navigation
* Smart search with suggestions
* Quick access to playlists and favorites
* Responsive mobile navigation

### 🎵 Advanced Playback Controls

* Play / Pause
* Next / Previous Track
* Shuffle & Repeat
* Interactive seek bar
* Volume controls
* Crossfade support

### 📜 Synchronized Lyrics

* Real-time lyric synchronization
* Karaoke-style highlighting
* Auto-scroll lyrics
* Click-to-seek lyric navigation

### ❤️ Playlist Management

* Create and manage playlists
* Favorite tracks
* Recently played history
* Personalized recommendations

### Immersive Visual Feedback
Audio Reactive Background

Music visualization reacts to:

Bass
Treble
Volume
Beat detection

Visual elements:

Animated waves
Floating particles
Neon rings
Pulsing gradients
